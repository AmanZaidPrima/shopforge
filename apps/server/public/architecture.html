Technology Decisions

Choice Why
Hono Identical code on Bun (dev) and CF Workers (prod). Zero adaptation layer.
Eta templates Compile once → JS function cached in memory. 0.005ms per render, vs 0.03ms+ for alternatives.
Alpine.js ~17KB Sprinkles on SSR HTML. No build, no hydration, auto-reinit after HTMX swaps.
HTMX ~17KB Navigation = body swap from server. Same render path as first load. No JS router.
Cloudflare KV Sub-ms reads at edge. Layout writes propagate globally in ~100ms.
TransformStream Enables progressive HTML streaming shown above. Browser starts before server finishes.
